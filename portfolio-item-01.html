Web App: Stock Price & News

Built using React and API queries

I created this app as an easy way to search for relevant news articles that may impact the price movement of a particular security. I still use this app almost daily!

The price data and news articles are sourced from TwelveData and GNews APIs, respectively. Data from each API query are stored in React state variables and passed into various display components in the app. The stock price chart utilizes a CanvasJS template.

Searchbar has built-in autocomplete search suggestions, and news article summaries link to the full article upon clicking.

Test the app yourself (hosted on GitHub Pages). Note that these are free developer APIs and the app may fail if the daily traffic limit is exceeded.
